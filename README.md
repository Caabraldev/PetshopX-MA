## 📌 Descrição
Sistema web de um petshop que apresenta produtos, serviços e permite o agendamento de atendimentos.
---
## 🛠️ Tecnologias utilizadas
* HTML
* CSS
* JavaScript
* Bootstrap
---
## 📁 Estrutura do projeto
```
petshop-mundo-animal
│
├── pages
│   ├── HTML
│   │   └── index.html
│   ├── CSS
│   │   └── style.css
│   └── Script
│       └── script.js
│
├── imagens
│   └── (imagens do projeto)
│
└── README.md
```
---
## 🚀 Como executar

1. Abra o projeto no VS Code
2. Vá até a pasta:
   ```
   pages/HTML
   ```
3. Abra o arquivo `onepage.html` no navegador
---
## 📄 Funcionalidades
* Exibição de produtos e serviços
* Formulário de agendamento
* Carrossel automático de imagens
* Navegação por seções
* Botões de contato
